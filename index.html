Mongodb Community edition nedlastnings Guide

Her har du en rask guide på hvordan sette opp MongoDB Community edition på en Linux Ubuntu maskin Du trenger: En Linux Ubuntu Maskin satt opp med internett tilkobling

Tutorial Video

Framgangs metode

Importer den offentlige nøkkelen som brukes av pakkebehandlingssystemet
sudo apt-get install gnupg curl
curl -fsSL https://www.mongodb.org/static/pgp/server-7.0.asc | \ sudo gpg -o /usr/share/keyrings/mongodb-server-7.0.gpg \ --dearmor
Opprett en liste-fil for MongoDB.
echo "deb [ arch=amd64,arm64 signed-by=/usr/share/keyrings/mongodb-server-7.0.gpg ] https://repo.mongodb.org/apt/ubuntu jammy/mongodb-org/7.0 multiverse" | sudo tee /etc/apt/sources.list.d/mongodb-org-7.0.list
Last inn den lokale pakkebasen på nytt.
sudo apt-get update
Installer MongoDB-pakkene.
sudo apt-get install -y mongodb-org
Start MongoDB
sudo systemctl start mongod
Sjekke MonoDB statusen
sudo systemctl status mongod
Nå har du satt opp en Mongo database på Ubuntu maskinen din
